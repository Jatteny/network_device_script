# 网络设备脚本
《网络设备脚本》的目标是以相同的代码以各种方式来控制不同的网络设备。

基本功能：
* 支持命令行控制网络设备（模拟人工敲命令）
* 支持网页控制网络设备（模拟人工点网页）

未来的设计目标：
* 支持软件定义网络
* 以其他方式控制网络设备

项目依赖项：
* [乘风龙王的代码库(python)](https://github.com/cflw/cflw_py)
